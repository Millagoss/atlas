# @atlas/ui

Shared UI component library and design system primitives.

## Responsibilities

- Reusable React components
- Design system tokens and primitives
- Consistent styling via Tailwind CSS

## Public API

- `Button`
- `ButtonProps`

## Dependencies

- `react` (peer)
- `react-dom` (peer)
- `@types/react` (dev)
- `@types/react-dom` (dev)
