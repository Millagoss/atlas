# @atlas/shared

Shared utilities, types, and constants used across all Atlas packages.

## Responsibilities

- Common TypeScript types (`AtlasMetadata`, `AtlasResult`, `AtlasError`)
- Utility functions (`generateId`, `noop`)
- Zero dependencies on engine packages

## Public API

- `generateId`
- `noop`
- `AtlasMetadata`
- `AtlasResult`
- `AtlasError`

## Dependencies

- None
