# Engineering Brief

**ID:** C05

**Title:** Spatial Scene Prototype

**Status:** Planned

---

# Objective

Implement the first complete spatial pipeline using deterministic data.

The goal is to prove that Atlas can transform an `ImageAsset` into a renderable `SpatialScene` without relying on AI.

This capability exists to validate the architecture, the scene builder, and the viewer integration independently of machine learning.

---

# Background

Atlas should be able to render a spatial scene regardless of where depth information originates.

For this capability, depth will be generated by a deterministic mock provider.

This creates a stable foundation for future AI providers.

---

# Requirements

Implement a `DepthProvider` abstraction.

Provide a default implementation:

- `MockDepthProvider`

The provider should generate predictable depth information suitable for testing.

Do not attempt to create realistic depth.

The output only needs to exercise the complete pipeline.

---

## SpatialScene Builder

Implement a builder that transforms:

ImageAsset
↓

DepthAsset
↓

SpatialScene

The SpatialScene should become the canonical scene representation used by the viewer.

Design it to support future expansion but avoid unnecessary complexity.

---

## Sandbox Integration

Update the Sandbox.

After image ingestion:

- Execute the mock depth provider.
- Generate a SpatialScene.
- Display a summary of the SpatialScene.

The summary may include:

- node count
- dimensions
- generated metadata
- processing time

No 3D rendering is required yet.

---

## Logging

Log:

- Mock depth generation
- SpatialScene creation
- Scene validation
- Pipeline completion

---

## Testing

Add automated tests for:

- MockDepthProvider
- SpatialScene Builder
- Scene validation
- Pipeline integration

---

# Constraints

Do not implement:

- AI inference
- ONNX Runtime
- WebGPU
- Three.js
- React Three Fiber
- Camera controls
- Rendering

---

# Acceptance Criteria

The capability is complete when:

- The pipeline generates a valid SpatialScene.
- The Sandbox displays the generated scene summary.
- Tests pass.
- Build succeeds.
- Public APIs are documented.

---

# Manual Verification

1. Upload an image.
2. Atlas creates an ImageAsset.
3. MockDepthProvider executes.
4. SpatialScene is generated.
5. Sandbox displays the scene summary.
6. No runtime errors occur.

---
